# Session Logs

This folder contains raw session exports from Rapsodo and TrackMan.

Rules:
- One file per session
- Do not edit files
- Do not aggregate here
- Filenames must include date, club, and device

Rapsodo note:
- CSV exports include footer rows like `Average` and `Std. Dev.`
- These rows are not shots and must be excluded during analysis

All interpretation happens elsewhere.
